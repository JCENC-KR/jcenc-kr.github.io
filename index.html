T.S.M — Tunnel of using Steel pipe Method

강관이 곧
구조물이 됩니다

다수의 강관을 압입하고 내부를 굴착해, H-BEAM과 철근콘크리트가 하나 된 합성 구조물을 만듭니다.

  • 최소 토피 시공 — 도로 표층 1m, 철도 노반 0.5m
  • 격벽철판 일체화 — 별도 방수재 없는 확실한 방수
  • 대구경 강관 D1,500~2,300 — 전 토층 적용
  • 선도관 조정장치 — 곡선 선형 R=60m 정밀 시공
EGR — Eco Grouting Rocket System

지반은 단단하게,
지하수는 깨끗하게

실리카졸계 저압 침투주입으로 지반을 고결하고 물길을 막는 특허 친환경 그라우팅입니다.

  • Gel-Time 수초~수분 조절 — 토질 맞춤 한정주입
  • 고결강도 20~60kg/㎠ — 반영구적 내구성
  • 해수·지하수 수질의 영향 없음
  • 용탈 최소화 — 지하수 오염 저감, 기존 SGR·LW 대비 개량
